
# anomaly-detection

Код и эксперименты магистерской диссертации **«Детектирование аномалий во временных рядах методами больших языковых моделей (LLM)»**.

**Автор:** Бабурина Анастасия Александровна  
**Научный руководитель:** Гущин Михаил Иванович  
**ВШЭ, ФКН, 2026**

### Структура репозитория

- **`synthetic/`** — эксперименты на синтетических данных  
  (генерация данных, обучение моделей, сравнение стратегий маскирования)

- **`msl/`** — эксперименты на реальном датасете MSL (NASA)

Репозиторий содержит Jupyter notebooks с основными экспериментами по четырём стратегиям маскирования:
- `random`
- `block`
- `channel_wise`
- `hybrid`

### Основные цели работы

- Исследовать, как разные стратегии маскирования влияют на качество детекции аномалий
- Выявить специализацию стратегий в зависимости от типа и длины аномалий
- Проверить переносимость результатов с синтетических данных на реальные (MSL)

### Технологии

- Python
- PyTorch
- Jupyter Notebook
- Transformer-based masked autoencoder (вдохновлено Time-RCD)

### Как использовать

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/a-nst-y/anomaly-detection.git
   cd anomaly-detection
   ```

2. Запускайте notebooks в папках `synthetic/` и `msl/`.
